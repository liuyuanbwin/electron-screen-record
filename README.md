#### 介绍
```
    基于 electron 实现的录屏软件, 鼠标拖动选择区域,录像完成选择视频保存位置,可直接打包exe安装文件
```

#### 安装
```
    yarn
```

#### 运行
```
    yarn start
```

#### 打包
```
    yarn dist-win
```

#### introduce
```
    record-screen by mouse drag,download video file by link
```
#### Instll
```
    yarn
```

#### run
```

```

#### package
